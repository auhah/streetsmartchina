## Start here

Use this hub for the phone setup pieces that can affect your first day in China: map access, map accuracy, and the data route you choose before arrival.

The first live problem is maps. Google Maps has separate access and accuracy issues in mainland China, while Apple Maps and Amap are the practical navigation choices.

## The Apps You Actually Need

Install the core apps before you fly, then finish registration and SMS checks while you still have fallback options.

| App | Install-order role |
| --- | --- |
| [Alipay](/money/alipay/) | Set up mobile payment before you need to scan a QR code. |
| [WeChat](/money/wechat-pay/) | Keep a second wallet path ready for merchants and trip messaging. |
| [Apple Maps or Amap](/connect/maps-apps/) | Choose your day-to-day map app before Google Maps accuracy becomes a problem. |
| [DiDi](/didi-in-china/) | Add the ride-hailing app before your first airport or station transfer. |
| [eSIM or local SIM plan](/connect/esim/) | Install before you fly so your phone has data on arrival. |

Install in this order: data plan first, maps second, payments third, ride-hailing fourth, and train tools after the basics work.

Secondary apps can wait until the core setup is done. Add a translation app with offline packs, because Google services depend on overseas routing. For train days, keep 12306 or Trip.com in the plan and use the [China trains guide](/trains/) for the route workflow.

For station navigation days, keep the [China trains guide](/trains/) handy.
